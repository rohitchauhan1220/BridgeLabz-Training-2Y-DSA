c) Logical Error in Algorithm B

The error is that the algorithm divides the sum by n inside a loop.
This causes the sum to be divided by n multiple times instead of only once.

Mathematical Principle Violated

The average formula:

Average= (Sum of values/n)

Algorithm B incorrectly performs:

Sum/n^n
